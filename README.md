#  Meu Projeto React

##  Sobre

Este é um projeto feito com React + Vite que mostra uma lista de perfis com nome, idade, email e foto.

##  Tecnologias usadas

* React
* Vite
* JavaScript
* CSS

##  Como rodar o projeto

```bash
# Instalar dependências
npm install

# Rodar o projeto
npm run dev
```

Depois acesse:
http://localhost:5173/

##  Funcionalidades

* Lista de perfis
* Exibição de nome, idade, email e foto
* Modo escuro (dark mode)

##  Estrutura

```
src/
 ├── App.jsx
 ├── App.css
 ├── main.jsx
```
